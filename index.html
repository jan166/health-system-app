하루 식단

"이건 먹지 마"가 아니라 "이걸 이 타이밍에 채워". 통밀빵·쌀밥 다 포함. 단백질 ~110g 고정, 탄수만 운동 유무로 조절.

칼로리 높아 보이지?

  • 내 유지칼로리 ~2,000kcal → 1,750/1,500이면 이미 하루 300~500 적자. 운동으로 더 태우면 충분히 빠짐.
  • BMR(1,338)은 "누워만 있을 때" → 실제 하루 소비는 BMR×활동 ≈2,000. BMR 밑으로 먹으면 대사 꺼짐(2024 실수).
  • 줄이는 건 정체 오면 Cycle 2~3에서 조금씩(→1,600/1,400). 미리 굶지 않기.

운동하는 날 ~1,750 kcal

단백질 110g탄수 190g지방 60g
  • 아침통밀빵 2장 + 계란 2 + 그릭요거트 (단백질 ~30g)
  • 점심쌀밥 150g + 닭가슴살/생선 + 채소 — 단백질 먼저
  • 운동전바나나 1 + 단백질 쉐이크 (탄수 몰기)
  • 운동후쌀밥 + 닭가슴살/소고기 + 시금치·브로콜리
  • 지방하루견과 1봉

쉬는 날 ~1,500 kcal

단백질 110g탄수 120g지방 60g
  • 아침계란 2 + 그릭요거트 + 냉동베리 (+ 통밀빵 1장)
  • 점심쌀밥 100g + 닭가슴살/생선 + 토마토·오이
  • 간식그릭요거트 or 삶은계란 2 (면·과자 대신)
  • 저녁흰살생선/닭 + 채소 넉넉히 + 쌀밥 소량
  • 지방하루견과 1봉

교환 자유 — 닭가슴살 100g = 소고기 80g = 생선 200g / 고구마 100g = 쌀 100g = 단호박 200g = 오트밀 50g. 질리면 바꿔 먹기.
